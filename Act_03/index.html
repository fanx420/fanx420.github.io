Activities

Activity 1
Webpage

Create an HTML website about a place in the world you would like to showcase (City, Country, Park, Mall, Mountain, Volcano, Tourist spot, Museum, etc). This may be one or multi-paged.

Activity 2
Webpage

Create a section of your homepage containing buttons/panels dedicated to your activities in this course. Transfer your activity 2 into a folder within and connect it to one of the buttons/panels.

Activity 3
Webpage

Create an HTML website that utilizes a free public API with JSON return. It can be about anything. It can have one or more pages
